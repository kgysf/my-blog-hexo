HTML Table-Layout Fixed 宽度设置无效

2022-04-18CC BY-NC-SA
在html中 当table的table-layout属性的值设置为fixed的时候, 如果第一行中存在跨列, 那么width将会失效 解决办法为在第一行输出一行无跨列行为的表格并隐藏
