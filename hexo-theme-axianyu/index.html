Axianyu

一个极简风格的Hexo主题, 基于adoubi主题, 修改了首页排版和少部分文章样式, 兼容手机端处理, 修改TOC为HEXO自带TOC, 请勿安装 hexo-toc; 并且确保Hexo版本 > 3.0

Installation

hexo-theme-adoubi developed with hexo 3.2  
hexo-theme-axianyu developed with hexo 6.0

需要的插件(dependencies):

  • hexo-renderer-pug
  • hexo-generator-feed

clone 主题到themes文件夹中, 改名为axianyu

git clone https://github.com/kgysf/hexo-theme-axianyu.git themes/axianyu

修改站点配置文件:

theme: axianyu

替换 themes/axianyu/sources/images/favicon.ico 为你自己的网站图标

upgrade

cd themes/axianyu
git pull origin master

部分说明

在文章中显示图片:

如果图片文件在站点的 source/images 中可使用 /images/yout-image.png 来访问

显示文章目录:

在文章的 Front-matter 部分增加 toc: true 即可显示目录. 更多关于 Front-matter 参考 Hexo doc: Front-matter

配置

全局配置文件 _config.yml 中的配置, 默认不包含以下配置, 可自己复制进去; 考虑主题更新影响, 没有选择使用theme config.

扩展 Front-matter 字段

rp: 4 # 1: 禁止转载; 2: 需要授权转载; 3: 需要注明来源转载; 4: 可随意转载
toc: true #是否显示目录, 不填默认不显示

rp默认为4, 可在全局配置中修改 `def_rp` 字段
